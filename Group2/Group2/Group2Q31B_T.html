அறிவியல்





Q1:ஒரு மாணவி புரத உணவைத் தவிர்த்து அதிகப்படியான கார்போஹைடிரேட் உணவை உட்கொள்கிறாள்.எந்த சத்து குறைபாட்டு நோய் அவளுக்கு ஏற்படும்







Q2: உறுதிப்படுத்துதல் ( A ) ; டயாபடிஸ் மெலிடஸ் நோயாளிகள் இரத்தத்தில் காணப்படும் கூடுதல் சர்க்கரை பயன்படுத்தப்படாமல் சிறுநீர் வழியாக வெளியேற்றப்படும்.காரணம் ( R ) ;  கணையம் போதுமான இன்சுலினை சுரப்பதில்லை







Q3: மெண்டல் தோட்டப்பட்டாணி ( பைசம் சட்டைவம் ) செடியில் 7 வகையான மாற்று உருவ வேறுபாடுகளைக் கண்டறிந்தார் கீழுள்ளவற்றில் ஒரு வகை வேறுபாடு மாறியுள்ளது.







Q4: ஆதி மனிதன் தோன்றியது……………………………………







Q5: கீழுள்ளவற்றில் எது பாரம்பரிய தன்மை கொண்டது ?







Q6: இயற்கை தேர்வு கோட்பாட்டினை வெளியிட்டவர்







Q7: உடற்செல் ஜீன் சிகிச்சை முறை என்பது







Q8: பட்டாணிச் செடியின் மஞ்சள் விதையின் பண்பானது,பச்சை நிற விதையின் மேல் ஓங்கு தன்மை கொண்டது.கீழ்கண்டவற்றுள் பச்சை நிற விதைக்கான ஜீனாக்கம்…………….







Q9: சில மனிதர்களின் நாவினை உருளச்செய்யும் ஓங்கு பண்பானது,உடல் குரோமோசோம்களால் கட்டுப்படுத்தப்படுகிறது.( நாவினை உருளச்செய்யபவர் = RR/Rr ; நாவினை உருளச்செய்ய இயலாதவர் = rr ) நாவினை உருளச்செய்யாத ஒரு சகோதரனும்,நாவினை உருளச்செய்யும் இரு சகோதரிகளும் உள்ளனர்.இவற்களின் பெற்றோர் இருவருமே நாவினை உருளச் செய்பவர்கள் எனில்,கீழ்கண்டவற்றில் பெற்றோர்களின் மரபணு வர்க்கம்…….







Q10: தொகுதி நிடேரியாவைச் சேர்ந்த பல செல் உயிரியான ஹைட்ராவில் பல வகையான இனப்பெருக்க முறை உள்ளது.கீழுள்ளவற்றில் புதிய சந்ததி குறிப்பிடும் படியான வேறுபாடுகளுடன் உருவாக்கப்படும் முறை…….
